# OpenPlot
一款开源的地理信息标绘组件

# v1.0
实现了基于 ArcGIS Engine + DevExpress 的客户端地理信息标绘组件，目前主要运用在防火指挥业务中，包括火场态势标绘、扑火战术标绘、队伍和物资调度、阻隔调度等。支持各元素的动态标绘、编辑、序列化反序列以及标注业务

主要实现的功能：
- 点：各类业务元素
- 线：直线、折线、弧线、自由线
- 面：多边形、自由面
- 箭标：内向钳形箭标、外向钳形箭标、多方向钳形箭标、行进方向等
- 标绘元素序列化与反序列化
- 标注的集成
